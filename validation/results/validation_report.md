生成时间: 2026-06-13

## 一、项目概述

pyOpenFOAM 是 OpenFOAM-13 的纯 Python/PyTorch 重实现，使用 PyTorch 作为张量后端，
支持 GPU 加速和端到端可微分模拟。OpenFOAM-13 参照源码位于 `.reference/OpenFOAM-13/`。

## 九、已知限制

1. **PISO 精度**: 惩罚方法不随网格缩放，需矩阵级 BC 处理
2. **OpenFOAM-13 参照**: Docker 已卸载，需重装或用 GCC 11+ 编译
3. **回流强度**: 粗网格过强（8x8: -0.608 vs -0.120）
4. **Poiseuille 流**: 稳态求解器发散
