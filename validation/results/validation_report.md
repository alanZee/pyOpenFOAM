## 3. 精度验证

### 3.1 Lid-Driven Cavity (Ghia et al., 1982)

**基准**: Ghia, K.N., Ghia, U., Shin, C.T. (1982). "High-Re solutions for incompressible flow using the Navier-Stokes equations and a multigrid method." *J. Comput. Phys.*, 48, 387-411.

| Re | 网格 | 求解器 | L2 相对误差 | 最大绝对误差 | 连续性误差 | 收敛迭代 |
|----|------|--------|------------|-------------|-----------|---------|
| 100 | 32x32 | SIMPLE | 13.4% | 0.106 | 8.8e-05 | 660 |
| 100 | 64x64 | SIMPLE | 6.2% | 0.053 | 9.7e-05 | 1309 |
| 100 | 128x128 | SIMPLE | 8.3% | 0.049 | 9.9e-05 | 1346 |

**分析**: Re=100 下 L2 误差随网格细化从 13.4% 降至 6.2%，但 128x128 反而升至 8.3%，表明存在网格相关的系统误差，可能来源于：
- 离散格式选择（一阶迎风 vs QUICK）
- 边界条件实现精度
- 压力-速度耦合算法收敛性

### 3.2 Couette 流

| 测量位置 | L2 相对误差 | 最大绝对误差 |
|----------|------------|-------------|
| 内部单元 | 0.001% | <1e-6 |
| 边界面 | 0.1% | <1e-3 |

Couette 流在内部单元达到极高精度，边界处误差略大。

### 3.3 Poiseuille 流

| 测量位置 | L2 相对误差 | 最大绝对误差 |
|----------|------------|-------------|
| 内部单元 | 0.02% | <1e-4 |
| 边界面 | 0.5% | <1e-2 |

### 3.4 高 Re Cavity (Re=400)

| 网格 | 松弛 U/p | 迭代 | 时间 | 连续性误差 | 状态 |
|------|----------|------|------|-----------|------|
| 32x32 | 0.2/0.1 | 500 | - | 2.8e-05 | 接近收敛 |
| 64x64 | 0.3/0.1 | 1000 | 1.4h | 3.8e-05 | 接近收敛 |
| 64x64 | 0.7/0.3 | 23 | 2.1min | NaN | 发散 |
| 128x128 | 0.7/0.3 | 5000 | 23.8h | 9.9e-03 | 未收敛 |
| 128x128 | 0.2/0.1 | - | 运行中 | - | 进行中 |

**分析**: Re=400 下标准松弛 (0.7/0.3) 导致发散，保守松弛 (0.2/0.1) 可维持稳定但收敛缓慢。这是 SIMPLE 算法在高 Re 数下的已知行为，需要更精细的网格或更高级的求解技术（如 PISO 或耦合求解器）。

## 8. 复现方法

```bash
# 安装
pip install -r requirements.txt
pip install -e .

# 运行全部测试
pytest tests/unit/ -q

# 运行验证
python validation/run_all.py

# 使用 OpenFOAM-13 Docker 镜像
docker load < openfoam13-compiled.tar.gz
docker run --rm -it openfoam13-compiled bash
```
